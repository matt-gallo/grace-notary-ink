# Hero Background Images

## Instructions

Place your hero background images in this folder with the following names:

- `hero-1.jpg` - First image (signing documents with notary stamp visible)
- `hero-2.jpg` - Second image (notary stamp on red pad with documents)
- `hero-3.jpg` - Third image (professional stamping document)

## Image Requirements

**Dimensions:**
- Recommended: 1920x1080px or higher
- Aspect Ratio: 16:9 (landscape)

**File Format:**
- JPG or JPEG (preferred for photos)
- File size: Optimize to under 500KB each for fast loading

**Image Style:**
- Professional notary scenes
- Neutral office environments
- Clean, not overly staged
- Good lighting
- Horizontal orientation

## How It Works

The hero section automatically rotates through these three images with a smooth fade effect:
- Each image displays for 5 seconds
- Smooth 2-second fade transition between images
- Light blue overlay maintains text readability
- Images loop continuously
